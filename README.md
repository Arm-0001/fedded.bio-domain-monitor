> ### Setup ✨
### Cloning the git repository
```
$ git clone https://github.com/catgir-ls/domains.git
$ cd domains
```
### Configuring
```
$ mv config.example.toml
$ vim config.toml
```
### Running 
```
$ deno run -A main.ts # Alternatively, you can run as a system service
```
